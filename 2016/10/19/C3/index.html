准备工具

  • phantom JS

    ​ PhantomJS是一个基于 WebKit 的服务器端 JavaScriptAPI。它全面支持web而不需浏览器支持，其快速，原生支持各种Web标准： DOM 处理, CSS 选择器, JSON, Canvas, 和 SVG。 PhantomJS 可以用于 页面自动化 ， 网络监测， 网页截屏，以及无界面测试 等。

    ​ 在这里phantom JS作为一个无界面浏览器，作为爬虫的工具用。无需安装，下载后解压即可。

  • selenium 命令行输入：pip install selenium

  • anconda2

    ​ anconda2是 Python科学技术包的合集，有自己的包管理工具 conda，同时支持pip工具，自带了很多常用的python的库，特别是一些很难安装的库。安装过程中，添加python到环境变量中去。

  • python

    ​ python2.7 是官方下载的python安装包，如果不想使用anaconda2，就安装这个，但装完之后，需要用pip去安装python库。添加python到环境变量中去。注意，如果安装了anaconda2就不用装这个，二选一

    ​ 如果选择官方python，所有使用到的库都要安装，也是使用pip安装 BeautifulSoup在一个库里面，安装BeautifulSoup：命令行行输入：pip install bs4

    总结与体会

    ​ 通过这次使用python爬虫爬取教务处成绩，首先第一感受到的是python的简洁与强大，代码只要短短几行，就能实现所需要实现的功能，它提供了许多我们可以调用使用的库，比如这次使用的selenium Beautiful soup还有其他的例如urllib cookielib等，这些库的提供对于我们方法的实现提供了很大的便利。但是同时还暴露出来的问题是python的运行时间有点长，这次爬虫爬取资源的响应速度也有点慢，可能这就是代码简单另一方面暴露出来的弊端。

    ​ 同时另一个让我感觉到简洁的就是Beautifulsoup库，在提取信息时 ，正则表达式也许可以实现相同的功能，但是内容相对来说就有点多，而且Beautifulsoup可以方便的提取出HTML标签中的内容，确实比较方便。

    ​ 总之，通过这次的作业，让我对Python,爬虫，正则表达式，等很多的 知识都有一个了解，但是同时暴露出来的自己技术能力不足的问题，刚从机电学院转过来，相当于一个电脑小白的我还有很多需要学习的地方，这次的作业应该是完成的最艰难的一次吧，并且绝大部分还是在帮助下进行的，认识到自己目前的水平，而且对这次内容掌握的并不熟练，一些语法并不是十分的熟悉，希望以后能多利用爬虫爬取数据练习，成为自己的一项技能。也尽快提高自己的知识面和技术水平。
