Unity3D 初级游戏开发

参考网站： 慕课网

应用工具

  • Unity 3D

    Unity3D是由Unity Technologies开发的一个让玩家轻松创建诸如三维视频游戏、建筑可视化、实时三维动画等类型互动内容的多平台的综合型游戏开发工具，是一个全面整合的专业游戏引擎。Unity类似于Director,Blender game engine, Virtools 或 Torque Game Builder等利用交互的图型化开发环境为首要方式的软件。

    其编程语言为C#和JS，在安装过程中，含有其自带的编辑器Monodevelope，用户可以选择打包下载，如果不喜欢的话，也可以配置自己喜欢使用的IDE。

编写过程

  • 资源包导入

    进入Unity3D 选择Assert->import pockage -> custom package..选择需要的资源包

  • 控制角色的移动

    Projece ->Script->右键选择creat -> C# Script

    自定义命名，这里命名为Player

    注意 脚本名字要与类名相同

    之后，将脚本文件直接拖拽到Player物体上面，或者点击Player -> inspector -> Add Component

  • 控制角色移动动画

    1. 动画控制机的建立

      • 打开 Project –> Object –> player->idle 双击move 将run forward 等四个组件全部拖拽到Animator中
      • 在图的右下角创建flout型 的 Speed 和 Angel 两个变量 初值均为0
      • 设置有向线段连接，并单击有向线段，在出现的inspect中设置不同中动画的调用

    2. 编辑动画脚本

      • 与之前提到的方法相同，右键Script->Create C# Script

      • 编辑脚本文件

        遇到的问题

        • 报错：You probably need to assign the move variable of The Player script in the inspector

          解决方案：在Player inspector 中的Move(script)的animator中添加Player

        • 报错：The referenced script on this behavior is missing

          解决方案：是由于脚本名称与类名不相同所致，改成相同即可

      感想

      ​ 通过这次用Unity3D对角色控制的体验，第一次感觉编写游戏离自己那么近，玩了这么多年游戏，也还是第一次用这样的方式控制角色的操作，充满新奇。但同时也体会到游戏开发也不是闹着玩的，必须有各方面的功底，最起码数学的要求还是有的，并不是像外界人所认为的那样不务正业，即使一个小小的控制移动这方面就非常的繁琐，可说起繁琐，Unity3D这个工具的使用相对来说又带来的巨大的便利，总之，这次课程拓宽了我的眼界，即使以后不从事游戏开发这个行业，也同样对这方面报有兴趣。
