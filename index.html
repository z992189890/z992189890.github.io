    开源的用于前端开发的工具包

    Bootstrap，来自 Twitter，是目前很受欢迎的前端框架。Bootstrap 是基于 HTML、CSS、JAVASCRIPT 的，它简洁灵活，使得 Web 开发更加快捷。它由Twitter的设计师Mark Otto和Jacob Thornton合作开发，是一个CSS/HTML框架。Bootstrap提供了优雅的HTML和CSS规范，它即是由动态CSS语言Less写成。

    • HTML

      JavaScript一种直译式脚本语言，是一种动态类型、弱类型、基于原型的语言，内置支持类型。它的解释器被称为JavaScript引擎，为浏览器的一部分，广泛用于客户端的脚本语言，最早是在HTML网页上使用，用来给HTML网页增加动态功能。

    Markdown语法的简要规则

    标题 在 Markdown 中，如果一段文字被定义为标题，只要在这段文字前加 # 号即可。

    列表 列表的显示只需要在文字前加上 - 或 * 即可变为无序列表，有序列表则直接在文字前加 1. 2. 3. 符号要和文字之间加上一个字符的空格。

    引用 只需要在文本前加入 > 这种尖括号（大于号）即可

    图片与连接 插入链接与插入图片的语法很像，区别在图片在最前方加一个 !号即可

    精神方面

    ​ 因为是第一次接触，比起来技术方面的收获，我觉得更有价值的是精神方面的内容

    1. 独立探索精神

      ​ 在授课时，学长讲的只有百分之六十，剩下的百分之四十都需要由我们自己学习，我知道，授之以鱼不如授之以渔，其实在听完课着手要做这个作业的时候，仍然是一头雾水，无从下手。但通过一遍遍摸索，一次次失败渐渐才有了头绪，慢慢的也逐渐清晰，到最后能够完成，这大大培养了自主学习能力和独立探索新事物的能力。

    2. 团队合作精神

      ​ 独立的同时，也要相信团队，积极与他人交流。在构建博客的过程当中，遇到了很多问题，很多问题都想要通过自己想出来，耗费了大量的时间与力气，到最后也没能解决。但是与别人交流之后，有别人指点迷津，很快得到了解决问题的方法。在遇到一些超出自己层次范围的问题时，还需积极与他人交流。

      还有一点最重要的就是细致吧，在配置Hexo的时候，仅仅是因为没有转到blog 目录下，一直配置完成不了，困惑了我一天多的时间。最后还是在问了别人之后，才发现问题，一个小问题，却带来很大的麻烦，编程更需要细致吧，一个字母的出错，就可能导致整个软件的失败。
